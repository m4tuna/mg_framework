Setting Usage
$grid-columns set the number of columns in your grid
$body-bg set color of body. can leave blank and specify elsewhere.
$full-width-header-mobile sets all columns within header element to 100% when true.
$full-width-main-mobile sets all columns within main element to 100% when true.
$full-width-tablet-portrait sets all columns to 100% on tablet portrait view when true.
$full-width-footer-mobile sets all columns within footer element to 100% when true.
$mobile-nav adds in a mobile nav when set to true.
$fixed-header sets the header to fixed. properly spaces main element below according to the height of the header.
$scroll-header changing header on window scroll.
$header-height set the height of the header.
