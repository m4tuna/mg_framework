grid

The grid-columns variable is used to set the size of the grid.

        $grid-columns: 12;

The grid for loop sets each column's width.

        @for $i from 1 through $grid-columns {
          .column-#{$i} {
            width: 100% / $grid-columns * $i;
          }
        }

mixins

breakpoints

        @mixin bp-small {
          @media only screen and (max-width: 30em) {
            @content;
          }
        }

The breakpoint mixins can be used inline to specifiy rules for mobile, tablet or large screen.

        .class {
           @include bp-small {
              foo: bar;
            }
         }

transitions

        @mixin transition-linear($secs) {
          -webkit-transition: linear $secs ease;
          -moz-transition: linear $secs ease;
          -o-transition: linear $secs ease;
          -ms-transition: linear $secs ease;
          transition: linear $secs ease;
        }

The transition mixins can be used inline to specifiy different transition animations. Options include linear, visibility, opacity, and all.

        .class {
           @include bp-small {
              foo: bar;
            }
         }

buttons

Using the @each directive below, buttons are set to a color map defined in _buttons.scss.

        @each $btn-color, $color in $btn-colors {
          .btn.#{$btn-color} {
            border-color:#{$color};
            &:hover {
              background: #{$color};
              color:$white;
            }
          }
        }

utilities

clearfix

        .class {
          @extend %clearfix;
        }

        #id {
          @extend %clearfix;
        }

The clearfix extend compiles to:

        .class, #id {
          *zoom: 1;
        }
        .class:before, #id:before, .class:after, #id:after {
          content: " ";
          display: table;
        }
        .class:after, #id:after {
          clear: both;
        }
